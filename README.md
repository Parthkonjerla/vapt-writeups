
How Attackers Use Nmap
A Cybersecurity Student's Breakdown — By K. Parth

About This Writeup
This writeup breaks down how attackers use Nmap during the 
reconnaissance phase of a penetration test — covering scan types, 
NSE scripts, what attackers look for in results, and how defenders 
can detect Nmap activity.

 What's Covered
- Scan types — Ping Sweep, SYN, TCP, UDP, Script Scan
- Nmap Scripting Engine (NSE) — vulnerability detection, 
  SMB enumeration, EternalBlue (MS17-010), SSH brute force
- What attackers look for in scan results
- How defenders detect and respond to Nmap scans

 Tools Used
`Nmap` `Kali Linux` `NSE Scripts` `VMware Lab`
